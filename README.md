
# Javascript Autocomment Bot

Provide the accounts and let the bot do the rest.


## Installation

Install with npm

```bash
  npm init
  npm i puppeteer
```

    
## Usage

Add the accounts and comments as many as you want in ` resource.json `

```json
{
    "accounts": [
        {
            "username": "",
            "password": ""
        },
        {
            "username": "",
            "password": ""
        }
    ],
    "comments": [
        "",
        ""
    ]
}
```

That's it, enjoy the automation!
